HeFQUIN Service API

HeFQUIN is a query federation engine for heterogeneous federations of graph data sources. This service provides a SPARQL endpoint for querying federated data.

Visit https://github.com/LiUSemWeb/HeFQUIN for additional info.

POST /sparql

Executes a SPARQL query based on the provided request body.

Request Headers

  • Content-Type: Required. Accepts application/sparql-query or application/x-www-form-urlencoded.
  • Accept: Supported MIME types:
    • application/sparql-results+json (default)
    • application/sparql-results+xml
    • text/csv
    • text/tab-separated-values

Request Body

  • If Content-Type is application/sparql-query, the body should contain the SPARQL query.
  • If Content-Type is application/x-www-form-urlencoded, the query should be included as a query parameter.

Response

  • 200 OK: Returns the query result with the specified MIME type.
  • 400 Bad Request: Returned if the SPARQL query is missing or empty.
  • 415 Unsupported Media Type: Returned if the Content-Type or Accept header is not supported.
  • 500 Internal Server Error: Returned if an error occurs during query execution.

GET /sparql

Executes a SPARQL query passed as a URL parameter.

Request Parameters

  • query: Required. The SPARQL query to be executed.

Request Headers

  • Accept: Supported MIME types:
    • application/sparql-results+json (default)
    • application/sparql-results+xml
    • text/csv
    • text/tab-separated-values

Response

  • 200 OK: Returns the query result with the specified MIME type.
  • 400 Bad Request: Returned if the SPARQL query is missing or empty.
  • 415 Unsupported Media Type: Returned if the Accept header is not supported.
  • 500 Internal Server Error: Returned if an error occurs during query execution.
